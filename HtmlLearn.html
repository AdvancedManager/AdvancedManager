Learn HTML

HTML is the foundation of every website. In this tutorial, you'll learn how to structure content, use tags effectively, and create well-formed web pages from scratch — step by step.

Start Learning

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to structure the content of web pages using elements known as tags.

Each tag has a specific role — for example, defining text, links, images, or layout sections. Tags usually come in pairs: an opening tag <tag> and a closing tag </tag>.

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

2. Basic HTML Document Structure

Every HTML page begins with a DOCTYPE declaration followed by the <html>, <head>, and <body> sections.

<!doctype html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome to HTML!</h1>
    <p>Learning HTML is fun.</p>
  </body>
</html>

The <head> section contains metadata, titles, and links to CSS or scripts, while the <body> holds the visible content.

3. Headings and Paragraphs

HTML offers six levels of headings: <h1> (most important) to <h6> (least important). Paragraphs are written inside the <p> tag.

<h1>Main Title</h1>
<h2>Subtitle</h2>
<p>This is a simple paragraph explaining headings in HTML.</p>

Headings help structure your content, improving both readability and SEO.

4. Links and Images

Use the <a> tag to create hyperlinks and the <img> tag to insert images.

<a href="https://example.com" target="_blank">Visit Example</a>
<br>
<img src="https://via.placeholder.com/150" alt="Placeholder Image">

The href attribute in the anchor tag defines the link destination, and alt in the image tag describes the image for accessibility.

5. Lists in HTML

Lists help organize information. There are three main types:

  • Ordered list — numbered (<ol>).
  • Unordered list — bulleted (<ul>).
  • Definition list — term and description (<dl>).
<h3>My To-Do List</h3>
<ul>
  <li>Learn HTML</li>
  <li>Practice CSS</li>
  <li>Build a Website</li>
</ul>

6. Tables

Tables organize data into rows and columns. Use <table>, <tr>, <th>, and <td> tags.

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>24</td>
  </tr>
</table>

7. Forms

Forms collect user input and send it to a server. Common elements include <input>, <label>, <textarea>, and <button>.

<form>
  <label>Name:</label>
  <input type="text" placeholder="Enter your name">
  <br>
  <button>Submit</button>
</form>

8. Semantic Elements

Semantic elements clearly describe their purpose in the document, improving readability and accessibility.

<header>Website Header</header>
<nav>Navigation Links</nav>
<main>Main Content</main>
<footer>Footer Information</footer>

9. Summary

Congratulations! You've learned the essential building blocks of HTML. You now understand how to create pages, format content, add media, and structure websites effectively.

Next, you’ll move on to CSS, where you’ll learn how to make your HTML pages visually beautiful and dynamic.
