Learn CSS step-by-step with visual examples and styled components. Build on your HTML knowledge to create beautiful, responsive web designs. This tutorial covers everything from selectors and box model to flexbox, typography, buttons, animations, and responsive design.

1. Introduction to CSS

CSS (Cascading Style Sheets) defines how HTML elements are displayed on screen, paper, or other media. It allows you to separate content (HTML) from presentation (CSS).

p { color: blue; font-size: 20px; }

By using CSS, you can control colors, spacing, fonts, layout, and much more, making your websites visually appealing.

2. CSS Selectors

Selectors target HTML elements so you can style them. Common types include:

  • Element selector: p { } — selects all <p> elements.
  • Class selector: .className { } — selects elements with the class.
  • ID selector: #idName { } — selects a specific element with an ID.
  • Attribute selector: [type="text"] { } — selects elements by attribute.
h1 { color: red; }
p { color: gray; }

3. The Box Model

Every element is a box with four parts:

  • Content: The actual text or image.
  • Padding: Space inside the box, around the content.
  • Border: Surrounds padding and content.
  • Margin: Space outside the border.
div {
  border: 1px solid black;
  padding: 10px;
  margin: 20px;
}

4. Colors and Backgrounds

Use color names, hex codes, or RGB values to style text and backgrounds:

body { background-color: #222; color: #fff; }
h1 { color: #ff8a00; }

Backgrounds can also be gradients, images, or patterns for a visually appealing design.

5. Typography

Typography controls the appearance of text:

  • Font family: font-family: Arial, sans-serif;
  • Font size: font-size: 18px;
  • Font weight: font-weight: 700;
  • Line height: line-height: 1.6;
  • Text alignment: text-align: center;
p { font-family:'Arial',sans-serif; font-size:18px; font-weight:600; line-height:1.6; text-align:center; }

6. Flexbox Layout

Flexbox is used for flexible and responsive layouts. Properties include:

  • display: flex
  • flex-direction: row | column
  • justify-content: space-between | center
  • align-items: center | stretch
  • flex-wrap: wrap
.container { display: flex; justify-content: space-between; align-items: center; }

7. Styling Buttons

Buttons can be styled for modern UI:

button { background: linear-gradient(180deg,#ff8a00,#ff6d00); color:white; padding:12px 20px; border-radius:10px; border:none; cursor:pointer; font-weight:700; }

8. Animations & Transitions

CSS animations and transitions add interactivity:

@keyframes example { 0% { transform: scale(1); } 50% { transform: scale(1.2); } 100% { transform: scale(1); } }
.animate { animation: example 2s infinite; }

9. Responsive Design

Responsive design ensures your website adapts to all devices. Techniques include:

  • Media queries: @media (max-width:768px) { ... }
  • Flexible units: %, em, rem instead of px
  • Flexible layouts using flexbox or grid
@media (max-width:768px) { .container { flex-direction: column; } p { font-size:16px; } }
